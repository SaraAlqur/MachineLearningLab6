# MachineLearningLab6
# Overview

This project uses Linear Regression to predict customer spending based on user activity data.

# Dataset

Includes features like Avg. Session Length, Time on App, Time on Website, Length of Membership, and Yearly Amount Spent (target). Non-numerical columns were excluded.

# Objective

To analyze customer behavior and predict yearly spending.

# Methodology

Data was explored, cleaned, and split into training and testing sets. A Linear Regression model was trained and evaluated.

# Results

The model shows a strong relationship between user activity and spending.

# Conclusion

Linear Regression effectively predicts spending, with Time on App and Length of Membership as key factors.
